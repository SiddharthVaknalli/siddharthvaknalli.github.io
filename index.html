Portfolio

Ultimate Chess

A rich Chess game for PC with single and multiplayer options! Incorporates an AI component to implement 4 levels of difficulty against the computer! Developed in Java using Eclipse. Heavily utilizes OOP principles and graphical components (Swing etc.) to model the game.

Priorify!

An Android app that inputs a list of tasks and prioritizes them according to criteria set by the user. Primarily developed a back-end algorithm in Java which prioritizes tasks by accounting for their due dates, importances, difficulties and time frames as set by the user. Utilized OOP principles and comparison and sorting techniques to implement the algorithm.

A creative cat-and-mouse 2D maze game with 3 game-play modes. Utilized OOP principles and data structures to model different game components and used file input to simulate maze. Designed graphics using Swing and AWT toolkits and implemented automated screen scrolling.
